EMV Tutorial

This EMV Tutorial will explain how to use the Smart Card Shell to explore an EMV chip card.

Overview:

1. Basics

1.1 Answer to Reset

Answer to Reset describes the first contact between card and terminal.

1.2 File Structure & APDU

Here we get some information about the file structure of EMV cards. Also we learn how the communication between card and terminal works.

1.3 ASN1 and TLV

This chapter describes the data structure

2. Transaction Process

2.1 Application Selection

Application Selection is the first function performed after the ATR. It selects the Master File of the card.

2.2 Initiate Application Process

The function initiates the transaction process and returns the AIP and AFL.

2.3 Read Application Data

The function reads all files which are listed in the AFL

2.4 Static Data Authentication

Static Data Authentication ensure the authenticity of the card's data.

2.5 Dynamic Data Authentication

DDA is a more securer way of authentication. It ensures additionally the uniqueness of cards
