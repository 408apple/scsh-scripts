EMV Tutorial - Part 1

This EMV Tutorial will explain how to use the Smart Card Shell to explore an EMV chip card.

Reseting the card

To reset an EMV card, we use the following script:

var card = new Card(_scsh3.reader);

var atr = card.reset(Card.RESET_COLD);

var atrbin = atr.toByteString();

print(atrbin);

print(atr.toString());
