Tutorial scripts

The tutorial directory contains a number of useful scripts:

  • stopwatch.js - Measuring time
  • loadbinaryfile.js - Load binary files
  • ctapidisplay.js - Display text on CT-API reader
  • ctapiverify.js - Verify PIN on CT-API reader
  • ctapi14443.js - Low level 14443 communication using ACG dual-iso reader

The tutorial/testing directory contains a simple test setup that shows how to use the testing facility.
