German GeldKarte

Scripts for the GeldKarte are located in the geldkarte directory directly underneath the installation directory (usually "c:\Program Files\CardContact\Smart Card Shell 3").

The following scripts are provided:

  • explore.js - Create outline view to explore the GeldKarte file structure
  • balance.js - Simple script to read and display the balance of the purse

You can run a script by entering load("geldkarte/<scriptname>"), e.g.

> load("geldkarte/explore.js");

or by selecting the script in the "File" / "Run Script" dialog.
