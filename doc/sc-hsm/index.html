SmartCard-HSM Public Support Scripts

The public SmartCard-HSM Support Scripts contain some examples and demos:

  • A scriptable CA to issue certificates for development and testing purposes
  • Two scripts to generate a PKCS#10 request and to store the certificate returned from the CA
  • A demo that shows the usage of a SmartCard-HSM in a Physical Access Control System (PACS)

This package provides a script to obtain a personal SSL certificate for access to the CardContact Developer Network (CDN).

The CDN contains additional information and scripts for SmartCard-HSM owner.

You can run the script by entering

>load("sc-hsm/ca/requestcert.js");

or by selecting the script in the "File" / "Run Script" dialog.

For details see the jsdoc
