OpenSCDP Script Collection

This collection of scripts can be used with the Smart Card Shell (SCSH3) or the Smart Card Scripting Environment For Eclipse (SSE4E).

Supported Applications

The Script Collection contains ready to run scripts for the following card applications:

  • German Identity Card (Neuer Personalausweis)
  • German Patient Data Card (eGK / Gesundheitskarte)
  • German Health Professional Card (HPC / Heilberufsausweis)
  • German Health Insurance Card (KVK / Krankenversichertenkarte)
  • Biometric Passport (MRTD) with Logical Data Structure (LDS)
  • Global Platform Card Manager (JavaCard)
  • MuscleCard Applet (JavaCard)
  • EMV Credit Card Application
  • German GeldKarte
  • Signature cards with PKCS#15 structure

Other useful Scripts

The script collection contains some Global Platform Profile examples in the profiles directory.

An example of running script fragments from Global Platform application profiles can be found in the gpscripting directory.

The tutorial directory contains further examples.
