OpenSCDP Script Collection

This collection of scripts can be used with the Smart Card Shell (SCSH3) or the Smart Card Scripting Environment For Eclipse (SSE4E).

Supported Smart Cards

The Smart Card Shell can be used with any smart card that conforms to ISO 7816-3.

The shell uses the OpenCard Framework as middleware to access smart cards. Extended functions can therefore be used with all smart cards for which an OCF card service implementation is available.

The default installation comes with a generic card service for cards compliant with ISO 7816, parts 4,5,8 and 9. The generic card service inspects the first historical byte in the ATR to determine the level of compliance with ISO 7816-4. If the historical bytes is '00' or '80', then full compatibility is assumed.

You can add your own OCF card service to the opencard.properties file. The card service must implement the FileAccessCardService interface.

The shell has been tested with

  • Starcos 3.x,
  • Micardo 2.x,
  • TCOS,
  • ACOS,
  • CardOS,
  • GemXCOS,
  • Seccos and
  • various Open Platform / Global Platform cards.

Supported Applications

The Smart Card Shell comes with ready to run scripts for the following card applications:

  • German Patient Data Card (eGK / Gesundheitskarte)
  • German Health Professional Card (HPC / Heilberufsausweis)
  • German Health Insurance Card (KVK / Krankenversichertenkarte)
  • Biometric Passport (MRTD) with Logical Data Structure (LDS)
  • Global Platform Card Manager (JavaCard)
  • MuscleCard Applet (JavaCard)
  • EMV Credit Card Application
  • German GeldKarte
